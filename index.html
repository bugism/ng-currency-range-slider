Ng Currency Range Slider

Wrapper AngularJS para o componente: Currency Range Slider .

Modo de uso

Importe o módulo nggs.currency-range-slider em seu app.

Exemplo
angular.module('yourApp', ['nggs.currency-range-slider']);

Os atributos de configuração do componente original foram transformados em atributos da diretiva.
O atributo on-range-change é o callback chamado quando há alteração de valores. Nele é passado o parâmetro $event que corresponde a um objeto contendo o novo intervalo selecionado({min, max}).

Exemplo
<ng-currency-range-slider
  ...
  on-range-change="rangeUpdated($event)">
</ng-currency-range-slider>

function rangeUpdated(newRange) {
  // do something when range is updated
}
Diretiva completa
<ng-currency-range-slider
  colors="{
    sliderBgColor,
    sliderBoxShadowColor,
    highlightBgColor,
    highlightBoxShadowColor,
    viewerBgColor,
    viewerColor,
    viewerInputBottomBorder
  }"
  ng-model="object"
  debounce-time="number"
  input-enter="boolean"
  label-prefix="string"
  limit-to-range="boolean"
  max="number"
  min="number"
  ng-change="callback function"
  range="{ min, max }"
  value-formatter="function reference"
  value-parser="function reference">
</ng-currency-range-slider>

Configuração

Para você não de passar sempre os mesmos parâmetros de configuração, você pode usar o Provider de configuração no qual você define uma configuração padrão e pode sobrescrever na diretiva quando desejar.

Exemplo
angular.module('yourApp', ['nggs.currency-range-slider'])
  .config(config);

config.$inject = ['ngCurrencyRangeSliderConfigProvider'];

function config(ngCurrencyRangeSliderConfig) {
  ngCurrencyRangeSliderConfig.setDefaults({
    colors,
    debounceTime,
    inputEnter,
    labelPrefix,
    limitToRange,
    max,
    min,
    valueFormatter,
    valueParser
  });
}
